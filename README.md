Q1:
What is the difference between getElementById, getElementsByClassName, and querySelector / querySelectorAll?
Ans: getElementById() selects one element by its unique id.
    getElementsByClassName() selects multiple elements by class name
    querySelector() select elements using any CSS selector and returns the first match
    querySelectorAll() select elements using any CSS selector returns all matches.
